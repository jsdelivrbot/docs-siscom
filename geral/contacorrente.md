## Geral

--:|---

**DVA** | Informe o dva. Por padrão, este campo é obrigatório.

**Descrição** | Informa o descrição.
**Raz.Social/Nome** | Informa o titular. Por padrão, este campo é obrigatório.

**Saldo** | Informa o saldo.
